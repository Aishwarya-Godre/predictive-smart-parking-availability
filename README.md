# 🚗 Predictive Smart Parking Availability System

## 📌 Project Overview

This project is a Machine Learning-based Smart Parking Availability Prediction System developed using Python and Flask. It predicts parking availability based on historical parking data and provides users with an easy-to-use web interface for booking parking slots.

---

## 🚀 Features

- User Registration & Login
- Machine Learning-based Parking Prediction
- Parking Slot Booking
- Admin Dashboard
- Parking Data Management
- Responsive Web Interface
- SQLite Database Integration

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite
- Machine Learning
- Scikit-learn

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Aishwarya-Godre/predictive-smart-parking-availability.git
```

2. Navigate to the project directory

```bash
cd predictive-smart-parking-availability
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```
